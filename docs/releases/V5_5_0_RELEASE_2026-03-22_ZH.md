# Deep-Sea Nexus v5.5.0 发布说明

发布日期：2026-03-22

## 摘要

`v5.5.0` 在现有 `runtime_middleware`、`execution_guard`、`capability_autotune_lab`
基础上，新增了一个**零侵入 Codex 记忆接入层**。

这一版不要求依赖 OpenClaw hook 才能抓取 Codex 记忆。系统现在可以按周期扫描本机
`~/.codex` 下的会话与历史数据，把摘要化结果增量写入 Memory v5。

## 本次包含

- 新增内部 `codex_periodic_ingest` 插件
  - 扫描 `~/.codex/sessions/**/*.jsonl`
  - 扫描追加型 `~/.codex/history.jsonl`
  - 维护增量 ingest state
  - 将 Codex 会话/历史摘要写入 Memory v5
- 新增手动运行入口：
  - `scripts/codex_periodic_ingest.py`
- 新增 Windows 计划任务安装入口：
  - `scripts/install_codex_periodic_ingest_task.py`
- 扩展 CLI 可观测性：
  - `health --json` 暴露 `plugins.codex_periodic_ingest.summary`
  - `paths --json` 暴露 Codex home、ingest workspace base、ingest state path、ingest metrics path、runner/installer path
- 默认配置新增 Codex 本机工作区写入支持：
  - `~/.codex/deepsea-nexus-workspace`

## 验证

- `tests/test_memory_v5.py` 新增 ingest / CLI / installer 定向覆盖
- 手动运行入口：
  - `python scripts/codex_periodic_ingest.py --json`
