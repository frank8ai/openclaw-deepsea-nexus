## Official Capability Contract

This release officially includes:

- sync compatibility API (`nexus_init / nexus_recall / nexus_add`)
- async plugin runtime lifecycle (`create_app()`)
- Memory v5 scoped storage (`MemoryScope / MemoryV5Service`)
- context-governed recall/inject/compress/rescue/replay loop
- evidence-gated durable decision writes
- lifecycle governance (`audit_lifecycle`, explicit archive, explicit backfill)
- local deployment and verification toolchain (`deploy / doctor / smoke / benchmark`)

## Governance Baseline

- fixed context rounds: `8 / 20 / 35`
- fixed trigger ratios: `0.7 / 0.85`
- canonical typed preserve fields:
  - `summary / goal / status / decisions / constraints / blockers / next_actions / questions / evidence / replay`
- durable decision gate:
  - no evidence pointer/replay pointer -> do not persist decision as durable memory
