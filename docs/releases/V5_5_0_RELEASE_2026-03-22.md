# Deep-Sea Nexus v5.5.0

Release date: 2026-03-22

## Summary

`v5.5.0` adds zero-intrusion Codex memory ingest on top of the existing
`runtime_middleware`, `execution_guard`, and `capability_autotune_lab`
baseline.

The new release does not require OpenClaw hooks for Codex memory capture.
Instead, it scans local Codex artifacts under `~/.codex` on a periodic basis
and writes digest-style durable memory into Memory v5.

## Included changes

- added internal `codex_periodic_ingest` plugin
  - scans `~/.codex/sessions/**/*.jsonl`
  - scans appended `~/.codex/history.jsonl`
  - tracks incremental ingest state
  - writes Codex session/history summaries into Memory v5
- added local runner:
  - `scripts/codex_periodic_ingest.py`
- added Windows scheduled-task installer:
  - `scripts/install_codex_periodic_ingest_task.py`
- extended CLI observability:
  - `health --json` exposes `plugins.codex_periodic_ingest.summary`
  - `paths --json` exposes Codex home, ingest workspace base, ingest state path, ingest metrics path,
    and runner/installer script paths
- updated default config to support Codex-local workspace writes under
  `~/.codex/deepsea-nexus-workspace`

## Validation

- focused ingest + CLI + installer coverage added to `tests/test_memory_v5.py`
- manual run path:
  - `python scripts/codex_periodic_ingest.py --json`
