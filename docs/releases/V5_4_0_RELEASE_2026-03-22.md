# Deep-Sea Nexus v5.4.0

Date: `2026-03-22`

## Summary

This release adds a report-first capability autotune lab for offline second-brain optimization, with the first tuning surface focused on runtime compression rules.

## Highlights

- added internal `capability_autotune_lab` plugin
- added configurable `runtime_middleware.compression.*` rules
- added offline experiment runner:
  - `scripts/capability_autotune_lab.py`
  - `scripts/capability_autotune_report.py`
- added repo-local compression golden eval pack:
  - `docs/evals/runtime_middleware_compression_golden_cases.json`
- exposed latest autotune report in:
  - `python -m deepsea_nexus health --json`
  - `python -m deepsea_nexus paths --json`

## Boundaries

- internal only, not public API
- report-first only
- no automatic runtime config mutation
- no automatic durable-memory mutation

## Validation

- targeted unittest coverage for configurable compression, CLI exposure, and autotune scripts
- full `python -m unittest tests.test_memory_v5 -v`
