# Deep-Sea Nexus v5.4.0 发布说明

日期：`2026-03-22`

## 摘要

这个版本新增了一个 report-first 的 `capability_autotune_lab`，用于离线优化第二大脑能力；首批优化面聚焦在 `runtime_middleware` 的压缩规则。

## 重点

- 新增内部 `capability_autotune_lab` 插件
- 新增可配置的 `runtime_middleware.compression.*`
- 新增离线实验脚本：
  - `scripts/capability_autotune_lab.py`
  - `scripts/capability_autotune_report.py`
- 新增 repo-local 压缩 golden eval：
  - `docs/evals/runtime_middleware_compression_golden_cases.json`
- `health --json` / `paths --json` 现在可以直接看到最近 autotune 报告

## 边界

- 仅内部能力，不是 public API
- 默认 report-first
- 不自动改线上 config
- 不自动改 durable memory

## 验证

- 已补充针对压缩规则配置、CLI 暴露、autotune 脚本的 unittest
- 全量验证命令：`python -m unittest tests.test_memory_v5 -v`
